# BITS stands for Bytebeat I T iS

## beats/
A place to save your bytebeats, comes with an intensive bytebeat tutorial (start.bytebeat)

## bytebeat.sh
Type `./bytebeat.sh "t t 8 >> &"` or any other expression and hear the result.
